# Impact-and-prediction-of-results-using-the-expected-goals-xG-metric
This dataset compiles comprehensive football data, including historical World Cup results (world_cup.csv), detailed match statistics from 1930 to 2022 (matches_1930_2022.csv)—some with 'Expected Goals' (xG) metrics—and the FIFA world rankings as of October 6, 2022 (fifa_ranking_2022-10-06.csv).
